# Coc Auto Tool

Bo cong cu Python ho tro tu dong hoa thao tac cho Clash of Clans qua ADB, gom:

- `gui.py`: giao dien ghi va phat lai macro cho thiet bi Android.
- `main.py`: script tu dong tan cong mau tren nhieu thiet bi.
- `server.py`: API Flask don gian de ghi nhat ky request/result vao SQLite.

## Yeu cau

- Python 3.10+
- ADB co san trong `PATH`
- Thiet bi Android hoac gia lap da bat che do debug

## Chay nhanh

```bash
python3 gui.py
python3 main.py
python3 server.py
```
